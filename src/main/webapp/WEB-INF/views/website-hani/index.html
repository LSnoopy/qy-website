农场简介
哈泥天空农场是一个以3-8周岁儿童为主角，以农场为载体的亲子自然教育项目。哈泥强调“learn by playing、learn by doing “的核心价值理念，以拟人化的原创卡通形象为故事主角，精心规划了各类农场情境项目，可以让孩子们与小动物嬉戏，与四季花草为伴，在快乐的玩耍中，充分释放孩子的自然天性，并通过各类生态认知、农耕渔猎体验、园艺手工艺等教育元素，以寓教于乐的方式提升儿童“手、脑、身、心”综合素质，收获爱与成就感。

哈蓓亲子游
哈蓓亲子游是一个以3-12周岁新中产阶级亲子家庭为主角的全新概念亲子游项目。哈蓓强调“learn by playing、learn by doing”的核心价值理念，以拟人化的原创卡通形象为故事主角，精心规划了各类农场情境项目，可以让孩子们与小动物嬉戏，与四季花草为伴，在快乐的玩耍中，充分释放孩子的自然天性，并通过各类生态认知、农耕渔猎体验、园艺手工艺等教育元素，以寓教于乐的方式提升儿童“手、脑、身、心”综合素质，收获爱与成就感。
动物巡游
哈泥天空农场有羊驼、矮脚马、牦牛、迷你猪、山羊等动物，并配备有专业的动物饲养人员。
合作的客户有银泰城、in77、龙湖天街、龙湖地产、印象城、恒隆广场、融创等。
农场主题美陈
哈泥天空农场有一支从事农场主题美陈设计和布置的专业软装团队。
合作的客户有龙湖天街、银泰城、印象城、龙湖地产。
